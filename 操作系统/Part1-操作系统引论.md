
### 1-1. 操作系统概念
1. **操作系统的地位和目标**
> 1. 操作系统的设计目标
>> - 方便性：提供用户接口， 使计算机系统更方便使用。
>> - 有效性：效管理和分配软硬件资源合理组织计算机工作流程来改善资源利用率。
>> - 可扩充性：适应计算机体系结构的发展。
>> - 开放性：适应不同厂家， 实现应用程序可移植性和互操作性。
2. **操作系统的作用**
> 1. 用户与计算机硬件系统之间的接口：
>> - 命令方式（os外壳）
>> - 系统调用方式（os内核）
> 2. 计算机系统资源的管理者：
>> - 管理对象
>> - 管理内容
> 3. 用作扩充机器（或虚拟机）
>> - 在裸机上添加处理机管理等， 使系统功能更强、使用更方便。
3. **操作系统的组成和层次模型**
> 1. 组成：
>> -  用户接口： 程序接口等。
>> - 操纵和管理有关对象的软件集合： pass。
>> - 操作系统对象： pass。
> 2. 层次模型 [底层模型流程图]

4. **操作系统的定义及实例**
> 1. 操作系统的定义:
>> - 操作系统是最基本的系统软件， 是一组**有效**管理和控制计算机硬件和软件资源、合理地对各类作业进行**调度**以控制系统工作流程， 方便用户使用计算机的程序集合。
> 2. 操作系统实例:
>> - MS DOS
>> - Windows
>> - UNIX
>> - Linux
>> - Novell Netware
>> - VxWorks
>> - AIX
>> - Android, IOS
